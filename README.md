# LeetCode Problems Solving

Python solutions to LeetCode problems, organized by problem number. Each solution comes with the original question, the code, and a short write-up covering the approach and time/space complexity.

## Structure

Each problem lives in its own folder under `solutions/`:

```
solutions/
└── 20-valid-parentheses/
    ├── question.txt   # original problem statement
    ├── solution.py     # Python solution
    └── README.md       # approach + complexity analysis
```
